# Sitio web de Adrián Cervantes

Sitio estático listo para publicar en Vercel.

## Publicar
1. Descomprime la carpeta.
2. Sube todos los archivos a un repositorio de GitHub.
3. En Vercel elige **Add New > Project** e importa el repositorio.
4. Framework Preset: **Other**.
5. Haz clic en **Deploy**.
6. En Vercel ve a **Settings > Domains** y agrega tu dominio de Namecheap.

La página principal es `index.html`.
